# Scoop Bucket for EarningsFeed

This is a [Scoop](https://scoop.sh) bucket for [EarningsFeed](https://earningsfeed.com) tools.

## Installation

```powershell
# Add the bucket
scoop bucket add earningsfeed https://github.com/earningsfeed/scoop-earningsfeed

# Install the CLI
scoop install earningsfeed
```

## Available Packages

| Package | Description |
|---------|-------------|
| [earningsfeed](https://earningsfeed.com) | CLI for the EarningsFeed API - SEC filings, insider transactions, and institutional holdings |
